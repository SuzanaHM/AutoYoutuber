Política de Privacidade — AutoYoutube

Versão 1.0 — Data de publicação: 20/09/2025

Aplicativo: AutoYoutube — uso pessoal com possibilidade de uso em múltiplas contas futuras.

1. Escopo

Esta Política de Privacidade descreve como o AutoYoutube coleta, armazena e usa informações no contexto do seu uso pessoal. O aplicativo integra-se com YouTube, TikTok e Google Cloud Text‑To‑Speech (TTS) para automatizar a repostagem de vídeos.

2. Controlador de Dados

Controlador: Usuária/Desenvolvedora: Suzana — contato: suzanahannahmartins@gmail.com.

3. Quais dados são coletados

O AutoYoutube não coleta dados de terceiros nem informações pessoais de outros usuários. Os dados tratados são:

  • Informações públicas do seu canal (quantidade de seguidores, número de vídeos, títulos/IDs dos vídeos) obtidas via API do YouTube mediante autorização.
  • Dados necessários para publicar no TikTok (tokens/identificador e permissões concedidas via OAuth).
  • Informações técnicas e logs locais para funcionamento (ex.: registros de execução armazenados localmente).

4. Como os dados são coletados

Os dados são obtidos diretamente das APIs do YouTube e do TikTok quando você autoriza o aplicativo via OAuth. O Google Cloud TTS é utilizado apenas quando você optar por gerar áudio via esse serviço.

5. Finalidades do tratamento

Os dados armazenados localmente são utilizados para:

  • Identificar e listar vídeos do seu canal YouTube para repostagem;
  • Publicar automaticamente (quando configurado) vídeos no TikTok usando as permissões concedidas;
  • Gerar áudio via Google Cloud TTS quando solicitado;
  • Monitoramento básico de funcionamento (logs) para depuração.

6. Base legal

Como aplicação de uso pessoal, o tratamento se baseia no seu consentimento e na execução do serviço por você solicitado (autenticação e ações via APIs). Você pode revogar autorizações diretamente nas plataformas (YouTube/TikTok), o que poderá interromper funcionalidades do aplicativo.

7. Compartilhamento com terceiros

O AutoYoutube não compartilha seus dados com terceiros. As únicas comunicações são diretas com as APIs do YouTube, TikTok e Google Cloud (para TTS) conforme as permissões que você concede nas telas de autorização dessas plataformas.

8. Armazenamento e retenção

Todos os dados e tokens são salvos localmente no seu computador. Permanecem armazenados enquanto o aplicativo estiver instalado ou até que você remova/exclua manualmente os dados. Logs de execução podem ser mantidos por até 12 meses localmente para fins de depuração, salvo se você optar por apagar antes.

9. Segurança

Como aplicação local, recomenda-se que seu computador esteja protegido (antivírus, atualizações de sistema). O aplicativo procura armazenar tokens de forma segura localmente; no entanto, sendo software pessoal, você é responsável pela segurança do ambiente onde o app roda.

10. Direitos do titular

Mesmo sendo aplicativo de uso pessoal, você pode solicitar alteração ou exclusão de dados locais. Para qualquer dúvida ou solicitação relacionada à privacidade, contate: suzanahannahmartins@gmail.com.

11. Cookies e rastreamento

O AutoYoutube não utiliza cookies no sentido de aplicações web remotas. Se utilizado um painel web local, poderá haver armazenamento local para preferências; não são enviados dados a terceiros para fins de tracking.

12. Dados de menores

O aplicativo não é direcionado a menores. Como uso pessoal, não há recolhimento intencional de dados de menores.

13. Alterações nesta política

Alterações serão registradas com nova data de publicação nesta mesma página e aplicadas a partir da publicação.

14. Contato
